MergeArrays - Объединение массивов
1
Создайте функцию, которая принимает два массива целых чисел и возвращает новый массив, содержащий элементы обоих массивов.
Пример:
MergeArrays([]int{1, 2, 3}, []int{4, 5, 6}) → []int{1, 2, 3, 4, 5, 6}
💡 Подсказка:

• Создайте пустой результирующий slice

• Используйте функцию append() для добавления элементов

• Сначала добавьте все элементы первого массива, затем второго

• Оператор ... поможет добавить все элементы сразу

🔍 Решение:
func MergeArrays(arr1 []int, arr2 []int) []int {
    var result []int
    result = append(result, arr1...)
    result = append(result, arr2...)
    return result
}
SplitArray - Разделение массива
2
Создайте функцию, которая принимает массив целых чисел и возвращает два новых массива: первый содержит элементы с четными индексами, второй - с нечетными.
Пример:
SplitArray([]int{1, 2, 3, 4, 5}) → ([]int{1, 3, 5}, []int{2, 4})
💡 Подсказка:

• Создайте два пустых slice для четных и нечетных индексов

• Переберите массив с помощью range

• Используйте оператор % для проверки четности индекса

• Если i % 2 == 0, то индекс четный

🔍 Решение:
func SplitArray(arr []int) ([]int, []int) {
    var evenIndexes, oddIndexes []int

    for i, value := range arr {
        if i%2 == 0 {
            evenIndexes = append(evenIndexes, value)
        } else {
            oddIndexes = append(oddIndexes, value)
        }
    }

    return evenIndexes, oddIndexes
}
ReverseArray - Разворот массива
3
Создайте функцию, которая принимает массив целых чисел и возвращает новый массив с элементами в обратном порядке.
Пример:
ReverseArray([]int{1, 2, 3}) → []int{3, 2, 1}
💡 Подсказка:

• Создайте пустой результирующий slice

• Используйте обратный цикл: идите от последнего элемента к первому

• Начните с индекса len(arr) - 1 и уменьшайте до 0

• Добавляйте каждый элемент в результат

🔍 Решение:
func ReverseArray(arr []int) []int {
    var result []int
    for i := len(arr) - 1; i >= 0; i-- {
        result = append(result, arr[i])
    }
    return result
}

9
Создайте функцию, которая принимает два целых числа (start и end) и возвращает массив, содержащий все целые числа в этом диапазоне включительно.
Пример:
CreateArrayWithRange(1, 5) → []int{1, 2, 3, 4, 5}
💡 Подсказка:

• Проверьте, что start не больше end

• Создайте пустой результирующий slice

• Используйте цикл от start до end включительно

• Добавляйте каждое число в результат

🔍 Решение:
func CreateArrayWithRange(start int, end int) []int {
    if start > end {
        return []int{}
    }

    var result []int
    for i := start; i <= end; i++ {
        result = append(result, i)
    }
    return result
}

• Переберите массив с получением индекса и значения

• Сравните каждый элемент с искомым значением

• Если нашли совпадение, верните текущий индекс

🔍 Решение:
func FindArrayElement(arr []int, element int) int {
    for i, value := range arr {
        if value == element {
            return i
        }
    }
    return -1
}

Factorial - Факториал числа
B4
Создайте функцию, которая принимает число N и возвращает его факториал (N!).
Пример:
Factorial(5) → 120  // 5! = 5*4*3*2*1 = 120
💡 Подсказка:

• Создайте переменную result с начальным значением 1

• Используйте цикл от 1 до N

• Умножайте result на каждое число

🔍 Решение:
func Factorial(n int) int {
    if n == 0 {
        return 1
    }
    result := 1
    for i := 1; i <= n; i++ {
        result *= i
    }
    return result
}
IsPrime - Проверка простого числа
B5
Создайте функцию, которая принимает число N и возвращает true, если число простое, и false в противном случае. Простое число - это число больше 1, которое делится только на 1 и на себя.
Пример:
IsPrime(7) → true   // 7 делится только на 1 и 7
IsPrime(8) → false  // 8 делится на 1, 2, 4, 8
IsPrime(2) → true   // 2 - простое число
💡 Подсказка:

• Числа меньше 2 не являются простыми

• Проверьте деление на все числа от 2 до √n

• Если найден делитель, число не простое

• Можно проверять только до i*i <= n

🔍 Решение:
func IsPrime(n int) bool {
    if n < 2 {
        return false
    }
    for i := 2; i*i <= n; i++ {
        if n%i == 0 {
            return false
        }
    }
    return true
}

CountDigits - Подсчет цифр в числе
B8
Создайте функцию, которая принимает целое число и возвращает количество цифр в нем.

CountDigits(12345) → 5

• Используйте цикл for с условием

• Считайте количество операций деления

• Обработайте случай числа 0

func CountDigits(n int) int {
    if n == 0 {
        return 1
    }
    if n < 0 {
        n = -n
    }
    count := 0
    for n > 0 {
        n /= 10
        count++
    }
    return count
}
ReverseNumber - Переворот числа
B9
Создайте функцию, которая принимает целое число и возвращает число с цифрами в обратном порядке.
Пример:
ReverseNumber(12345) → 54321
ReverseNumber(100) → 1
ReverseNumber(7) → 7
💡 Подсказка:

• Используйте цикл для извлечения цифр справа налево

• Получайте последнюю цифру через n % 10

• Стройте результат: result = result*10 + digit

• Убирайте последнюю цифру: n = n / 10

🔍 Решение:
func ReverseNumber(n int) int {
    if n < 0 {
        return -ReverseNumber(-n)
    }
    
    result := 0
    for n > 0 {
        digit := n % 10
        result = result*10 + digit
        n = n / 10
    }
    return result
}

        return "B"

        return "C"
    } else if score >= 60 {

Slice Quiz 1 - Базовый append
S1
Проанализируйте код и определите, что произойдет с исходным слайсом:
Код:
arr := []int{1, 2, 3}
fmt.Printf("len=%d, cap=%d\n", len(arr), cap(arr)) // len=3, cap=3

💡 Объяснение:

• Исходный слайс имеет capacity = 3, что равно его длине

• Исходный слайс остается неизменным: [1, 2, 3]

• Новый слайс получает копию данных + новый элемент: [1, 2, 3, 4]

Slice Quiz 9 - Capacity Doubling
S9
Изучите, как Go увеличивает capacity при росте слайса:
Код:
s := []int{1, 2}               // len=2, cap=2
fmt.Printf("Initial: len=%d, cap=%d\n", len(s), cap(s))

s = append(s, 3)               // что с capacity?
fmt.Printf("After 1: len=%d, cap=%d\n", len(s), cap(s))

s = append(s, 4, 5)            // что теперь?
fmt.Printf("After 2: len=%d, cap=%d\n", len(s), cap(s))
Какие значения capacity будут выведены?
💡 Объяснение:

• Go удваивает capacity при превышении для маленьких слайсов

• Начальная capacity = 2

• При добавлении 3-го элемента: capacity = 2 * 2 = 4

• При добавлении 4-го и 5-го элементов: нужна capacity ≥ 5

• Go снова удваивает: 4 * 2 = 8

Slice Quiz 10 - Nil vs Empty
S10
В чем разница между nil слайсом и пустым слайсом?
Код:
var nilSlice []int
emptySlice := []int{}
madeSlice := make([]int, 0)

fmt.Printf("nil: len=%d, cap=%d, nil=%t\n", 
    len(nilSlice), cap(nilSlice), nilSlice == nil)
fmt.Printf("empty: len=%d, cap=%d, nil=%t\n", 
    len(emptySlice), cap(emptySlice), emptySlice == nil)
fmt.Printf("made: len=%d, cap=%d, nil=%t\n", 
    len(madeSlice), cap(madeSlice), madeSlice == nil)
Какие значения nil будут выведены?
💡 Объяснение:

• nil слайс: не имеет underlying array, pointer = nil

• []int{}: пустой слайс с underlying array, pointer ≠ nil

• make([]int, 0): тоже создает underlying array, pointer ≠ nil

• Все имеют len=0, cap=0, но только nil слайс равен nil

• append() работает одинаково со всеми тремя
