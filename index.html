MergeArrays - Объединение массивов
1
Создайте функцию, которая принимает два массива целых чисел и возвращает новый массив, содержащий элементы обоих массивов.
Пример:
MergeArrays([]int{1, 2, 3}, []int{4, 5, 6}) → []int{1, 2, 3, 4, 5, 6}
💡 Подсказка:

• Создайте пустой результирующий slice

• Используйте функцию append() для добавления элементов

• Сначала добавьте все элементы первого массива, затем второго

• Оператор ... поможет добавить все элементы сразу

🔍 Решение:
func MergeArrays(arr1 []int, arr2 []int) []int {
    var result []int
    result = append(result, arr1...)
    result = append(result, arr2...)
    return result
}
SplitArray - Разделение массива
2
Создайте функцию, которая принимает массив целых чисел и возвращает два новых массива: первый содержит элементы с четными индексами, второй - с нечетными.
Пример:
SplitArray([]int{1, 2, 3, 4, 5}) → ([]int{1, 3, 5}, []int{2, 4})
💡 Подсказка:

• Создайте два пустых slice для четных и нечетных индексов

• Переберите массив с помощью range

• Используйте оператор % для проверки четности индекса

• Если i % 2 == 0, то индекс четный

🔍 Решение:
func SplitArray(arr []int) ([]int, []int) {
    var evenIndexes, oddIndexes []int

    for i, value := range arr {
        if i%2 == 0 {
            evenIndexes = append(evenIndexes, value)
        } else {
            oddIndexes = append(oddIndexes, value)
        }
    }

    return evenIndexes, oddIndexes
}
ReverseArray - Разворот массива
3
Создайте функцию, которая принимает массив целых чисел и возвращает новый массив с элементами в обратном порядке.
Пример:
ReverseArray([]int{1, 2, 3}) → []int{3, 2, 1}
💡 Подсказка:

• Создайте пустой результирующий slice

• Используйте обратный цикл: идите от последнего элемента к первому

• Начните с индекса len(arr) - 1 и уменьшайте до 0

• Добавляйте каждый элемент в результат

🔍 Решение:
func ReverseArray(arr []int) []int {
    var result []int
    for i := len(arr) - 1; i >= 0; i-- {
        result = append(result, arr[i])
    }
    return result
}
MultiplyArray - Умножение элементов
4
Создайте функцию, которая принимает массив целых чисел и множитель, возвращает новый массив с каждым элементом, умноженным на множитель.
Пример:
MultiplyArray([]int{1, 2, 3}, 2) → []int{2, 4, 6}
💡 Подсказка:

• Создайте пустой результирующий slice

• Переберите каждый элемент исходного массива

• Умножьте каждый элемент на заданный множитель

• Добавьте результат умножения в новый массив

🔍 Решение:
func MultiplyArray(arr []int, factor int) []int {
    var result []int
    for _, value := range arr {
        result = append(result, value*factor)
    }
    return result
}
FilterArrayByEven - Фильтр четных чисел
5
Создайте функцию, которая принимает массив целых чисел и возвращает новый массив, содержащий только четные числа.
Пример:
FilterArrayByEven([]int{1, 2, 3, 4}) → []int{2, 4}
💡 Подсказка:

• Создайте пустой результирующий slice

• Переберите каждый элемент массива

• Проверьте четность числа с помощью value % 2 == 0

• Добавляйте в результат только четные числа

🔍 Решение:
func FilterArrayByEven(arr []int) []int {
    var result []int
    for _, value := range arr {
        if value%2 == 0 {
            result = append(result, value)
        }
    }
    return result
}

GetArraySum - Сумма элементов
7
Создайте функцию, которая принимает массив целых чисел и возвращает сумму всех его элементов.
Пример:
GetArraySum([]int{1, 2, 3}) → 6
💡 Подсказка:

• Создайте переменную для хранения суммы (начальное значение 0)

• Переберите все элементы массива

• Добавляйте каждый элемент к общей сумме

• Верните итоговую сумму

🔍 Решение:
func GetArraySum(arr []int) int {
    sum := 0
    for _, value := range arr {
        sum += value
    }
    return sum
}
GetLastArrayElement - Последний элемент
8
Создайте функцию, которая принимает массив целых чисел и возвращает последний элемент массива.
Пример:
GetLastArrayElement([]int{1, 2, 3}) → 3
💡 Подсказка:

• Проверьте, что массив не пустой

• Используйте len(arr) для получения длины

• Последний элемент находится по индексу len(arr) - 1

• Не забудьте обработать случай пустого массива

🔍 Решение:
func GetLastArrayElement(arr []int) int {
    if len(arr) == 0 {
        return 0
    }
    return arr[len(arr)-1]
}
CreateArrayWithRange - Создание диапазона
9
Создайте функцию, которая принимает два целых числа (start и end) и возвращает массив, содержащий все целые числа в этом диапазоне включительно.
Пример:
CreateArrayWithRange(1, 5) → []int{1, 2, 3, 4, 5}
💡 Подсказка:

• Проверьте, что start не больше end

• Создайте пустой результирующий slice

• Используйте цикл от start до end включительно

• Добавляйте каждое число в результат

🔍 Решение:
func CreateArrayWithRange(start int, end int) []int {
    if start > end {
        return []int{}
    }

    var result []int
    for i := start; i <= end; i++ {
        result = append(result, i)
    }
    return result
}
FindArrayElement - Поиск элемента
10
Создайте функцию, которая принимает массив целых чисел и элемент для поиска. Возвращает индекс элемента в массиве или -1, если элемент не найден.
Пример:
FindArrayElement([]int{1, 2, 3}, 2) → 1
💡 Подсказка:

• Переберите массив с получением индекса и значения

• Сравните каждый элемент с искомым значением

• Если нашли совпадение, верните текущий индекс

• Если цикл завершился без находок, верните -1

🔍 Решение:
func FindArrayElement(arr []int, element int) int {
    for i, value := range arr {
        if value == element {
            return i
        }
    }
    return -1
}
